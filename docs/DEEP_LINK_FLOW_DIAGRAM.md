## Quick Reference

### Key Components

- **DeepLinkHandler**: Processes incoming URLs
- **AuthContext**: Manages auth state
- **Supabase Client**: Handles token exchange
- **SessionManager**: Monitors timeouts

### Key Functions

- `Linking.getInitialURL()`: Get URL that opened app
- `Linking.addEventListener()`: Listen for URLs while running
- `supabase.auth.setSession()`: Exchange tokens
- `supabase.auth.onAuthStateChange()`: Listen for auth events

### Key States

- `isProcessingLink`: Deep link being processed
- `isAuthenticated`: User logged in
- `isLoading`: Initial session check
- `linkError`: Deep link error message
